# Интеграция ИС КИПР и ИС УИП (Адепт)

## Общее сведения об интеграции 

### 1.1. Бизнес-контекст и цели
Данный репозиторий описывает архитиктуру для реализации двустороннего взаимодействия  между системами:

*   **ИС КИПР** — Информационная система формирования консолидированной инвестиционной программы ГрКИПРы РусГидро (Система-приемник).
*   **ИС УИП (Адепт)** — Информационная система управления инвестиционными проектами (Мастер-система, источник данных).
*   **ЕИП** — Единая интеграционная платформа ООО «РусГидро ИТ сервис» (Интеграционная шина, транспортный уровень).

**Бизнес-цель:** Автоматизация обмена данными для передачи данных по Проекту, Контрольным точкам проекта, Паспорту проекта. 

### 1.2. Архитектура интеграции
Интеграция реализована по следющим принципам:
*   **Тип взаимодействия:** Асинхронный.
*   **Способ передачи:** Файловый обмен.
*   **Формат данных:** Excel-файлы.
*   **Транспорт:** SFTP-сервер ЕИП.
